## Laravelの教科書 第7章: 投稿データの作成と保存

### 学んだこと

**投稿データの作成と保存**
- フォームの作成(Blade)
- データの受け取り(Controller)
- バリデーション(入力チェック)
- データベースへの保存(Model)

**フォーム作成**
```blade
<form method="POST" action="/posts">
    @csrf
    <input type="text" name="title">
    <button type="submit">送信</button>
</form>
```

**バリデーション**
```php
$validated = $request->validate([
    'title' => 'required|max:255',
]);
```

**データ保存**
```php
Post::create($validated);
```

### 気づき

- PHPのフォーム処理($_POST)がLaravelだと簡潔
- バリデーションが簡単に書ける
- CSRFトークン(@csrf)の重要性を理解
